# Weather App

- Search for any location or adress to get weather data
- Get current weather data, along with average values for the week
- Quickly tell the conditions of any given day in the week
- Click a day in the next week to see more details

Live Link: https://spyrospolemis.github.io/Weather-App/

# Tech Stack:

- HTML, CSS, JS
- Webpack
- [date-fns](https://date-fns.org/) library for formatting dates
- VisualCrossing [Weather API](https://www.visualcrossing.com/weather-api/) free tier, for weather data

# Project Goals:

- Use a 3rd party API
- Practice using asynchronous code/functions

All AI usage declared in comments.
